# MyRasaAssistant
Agent conversationnel s'appuyant sur RasaPro


# Install
# Rasa supports Python 3.10 and 3.11
**générer une clé ssh
ssh-keygen -t ed25519 -C "ton-email@exemple.com"

git clone https://github.com/Cyberquoi/MyRasaAssistant.git

pip install rasa-pro

- rasa train
- rasa inspect
- rasa actions
