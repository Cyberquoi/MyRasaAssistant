# MyRasaAssistant
Agent conversationnel s'appuyant sur RasaPro


# Install



# Créer le projet (Rasa supports Python 3.10 and 3.11)
git clone https://github.com/Cyberquoi/MyRasaAssistant.git

## créer l'environnement virtuel

pip install rasa-pro

# Commandes Rasa
- rasa train
- rasa inspect
- rasa actions
